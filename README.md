## :rocket: Setup

### Step 1: Clone/Fork the repo

To get started, clone/fork the repo

```sh
git clone https://github.com/sleithdylan/sneakerz.git
```

### Step 2: Install Dependencies

Change directory

```sh
cd sneakerz
```

Next, type the following in the root of this project:

```sh
npm install
```

### Step 3: Create build directory

Create build directory

```sh
npm run build
```

### Step 4: Start the local server

Change directory

```sh
cd backend/
```

To start the local server, type the following:

```bash
node server.js
```

### Step 5: Open application in browser

Open link when server is running: [http://localhost:4000/](http://localhost:4000/)
